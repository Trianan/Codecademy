Container Properties

Property Value Description

width height

Value in px, em, percentage, or keyword Sets the dimensions of the content area if box-sizing property is set to content-box, or the border area if box-sizing is set to border-box.

padding

Value in px, em, or percentage. Controls the space between the content and the margin.

border

Width in px, rem, or keyword; style keyword; color. Sets the width, style, and color of the border around an element.

margin

Size in px, em, percentage or auto. Sets the space between an elements border and other elements.

box-sizing

content-box or border-box. Controls whether width/height take into account the added space from padding and border width. Border-box sized elements total area will be equal to the specified dimensions. Content-box will only control the size of the content itself; the padding and border will add onto the specified size.

Layout Properties

Property Value Description

position

relative Positions element relative to its parent.
absolute Positions element relative to the entire page.
fixed Positions element relative to initial container, but makes it stay in that position regardless of scrolling.
sticky Positions element as normal, but keeps element at defined position when the viewport scrolls away from its parent element.

display

block The element may have a defined width and height, and will appear on its own line.
inline The element will only take up as much space as its content, and may appear on the same line as other elements.
inline-block The element may have a defined width and height, and may appear on the same line as other elements.
none The element is removed from the flow of the document as if it doesnt exist; the space reserved for the element is removed.

z-index

An integer number. Defines what order elements will render over eachother; lower z-indices will render below higher ones.

float

left, right, both, none Moves an element to the specified side of its container.

clear

left, right, both, none Determines if an element is moved below a preceding floating element; for example, if text is meant to clear an element, it will move the entire text element below the floating element instead of flowing around it.

Color-Modifying Properties

Property Value Description

color

Sets color of foreground content.

background-color

Sets color of background behind content.

 keyword colors (e.g. midnightblue) Sets color to one of the named color values.
hexidecimal value (e.g. #FF8016) Sets color as defined by a hexidecimal value; each pair of digits corresponds to red, green, and blue values respectively.
RGB value (e.g. rgb(255, 0, 0)) Sets colour according to an RGB-defined value.
RGBA value (e.g. rgba(255, 50, 65, 0.5)) Same as RGB, but includes parameter for alpha/opacity value between 0 and 1.
HSL value (e.g. hsl(220, 75%, 50%)) Sets a color defined by its hue (value between 0 and 360), and two percentages which defined the saturation and lightness of the color.
HSLA value (e.g. hsla(345, 80%, 20%, 0.25)) Same as HSL, but includes parameter for alpha/opacity value between 0 and 1.

Typographical Properties

Property Value Description

font-family

Font names (e.g. "Times New Roman", serif)) Sets the typeface of text in the affected element.

font-weight

Keywords (bold, lighter) or numbers between 0-1000 Sets how thick/bold the given text appears.

letter-spacing

Values in terms of px or em. Sets the spacing between individual letters in the affected text.

word-spacing

Values in terms of px or em. Sets the spacing between words in the affected text.

line-height

A unitless numerical scaling factor based on the initial spacing. Scales the default spacing between lines of affected text by a given scaling factor. Values above 1 will increase the spacing, for example.
